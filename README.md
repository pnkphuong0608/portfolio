Objective: Create an interactive dashboard that enables managers to monitor supply chain performance and identify opportunities for cost reduction, quality improvement, and operational optimization.

Tools Used: Microsoft Excel, Pivot Tables, Pivot Charts, Slicers, Conditional Formatting, KPI Cards

Key Insights Covered:
Sales and revenue performance, Inventory availability and stock monitoring, Supplier lead time and defect analysis, Transportation cost and logistics efficiency, Quality control performance, Product profitability analysis

This dashboard demonstrates the application of Excel-based business intelligence techniques to support strategic and operational decision-making.
